# 🎧 Spotify Music Analytics Dashboard

A comprehensive data analytics dashboard built to analyze Spotify songs, artists, popularity trends, and release patterns using interactive visual storytelling.

This project transforms raw Spotify metadata into actionable insights for understanding music consumption, artist performance, and content trends across time.

# 📌 Project Overview

This dashboard provides an end-to-end analytical view of Spotify’s music ecosystem, enabling users to:

- Analyze artist popularity and output

- Track song-level performance

- Compare explicit vs non-explicit content

- Understand release trends by year and month

- Evaluate song duration, popularity, and positioning metrics

The UI is inspired by Spotify’s native design language to improve usability and familiarity, while maintaining a strong analytical backbone.

# 🧠 Key Insights Delivered

Top-performing artists and songs based on popularity

Content distribution by album type (single, album, compilation)

Temporal trends in music releases and popularity

Explicit content analysis across the dataset

Average duration and popularity metrics by artist and song
